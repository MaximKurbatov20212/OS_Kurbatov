1) Где хранится текущая директория для данного процесса?

Ответ: Текущая директория - это атрибут процесса, которые хранятся или на стеке или в ядре. Данный атрибут хранится в ядре.

2) В каком порядке будет происходить поиск исполняемого файла при вызове execvp?

Ответ: На семинаре я сказал неверную информацию, что будет происходить поиск в текущей директории. Это не так, по умолчанию этого
не будет происходить, но можно добавить текущую директорию в PATH, и тогда будет происходить поиск, но так делать не рекомендуется.

Поиск будет происходить следующем образом:
    1) Поиск по директориям, указанным в PATH
    2) В директории /usr/bin
